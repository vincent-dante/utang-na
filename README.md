https://utangna.vercel.app/

## Tech Stacks

## Features

- Search & Filters for loans.
- UI - Landing Page, Loans Page, Application Form.
- Mobile Responsive.

## Build Setup

To Run this project locally.

1. Clone the project.
2. yarn install
3. yarn dev -o
